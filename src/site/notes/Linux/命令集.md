# 快捷键

- `Ctrl + u` ：删除光标前面的内容
- `Ctrl + k` ：删除光标向后的内容
- `Ctrl + a` ：移动光标到最前面
- `Ctrl + e` ：移动光标到最后面
- `Ctrl + 左右方向键` ：移动每个单词首/尾部
